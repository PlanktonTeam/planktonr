Data Quality Information:

Scope of quality information: dataset

Lineage:
The Commonwealth Marine Regions has been created using a combination of:
- 2006 Australian Maritime Boundaries (AMB).
- Large Marine Domains
- IMCRA v4.0 - Provincial Bioregions
- AUS100k (for some island coastlines)
- CAPAD 2004
- Torres Strait Fisheries boundaries

Source data acknowledgements:
Commonwealth Marine Regions (c) Commonwealth of Australia, Australian Government Department of the Sustainability, Environment, Water, Population and Communities, 2012

Data quality report - Absolute external positional accuracy:
The data from which this dataset was compiled have scales varying between 1:100,000 and approximately 1:2,500,000, giving a locational accuracy varying between 100-2,500 metres. Most line work is from the AMB dataset which has a scale of 150,000.

Data quality report - Attribute accuracy:
Attributes include ENVIRON (Marine or Land), REGION (name of marine planning region) and AREA_KM2 (area in square kms).

Data quality report - Conceptual consistency:
The dataset was constructed using the latest AMB, GBRMP and officially endorsed "internal" boundaries.

Data quality report - Completeness:
Spatial and attribute data is complete for all of Australia.
