# Get taxa accumulation data for plotting

Get taxa accumulation data for plotting

## Usage

``` r
pr_get_TaxaAccum(Survey = "NRS", Type = "Zooplankton")
```

## Arguments

- Survey:

  "NRS or "CPR"

- Type:

  "Zooplankton" or "Phytoplankton"

## Value

A data frame

## Examples

``` r
df <- pr_get_TaxaAccum(Survey = "NRS", Type = "Zooplankton")
```
