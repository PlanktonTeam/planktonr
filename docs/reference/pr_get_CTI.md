# Calculate Community Temperature Index (CTI) for plankton samples

Calculate Community Temperature Index (CTI) for plankton samples

## Usage

``` r
pr_get_CTI(Type = "Zooplankton")
```

## Arguments

- Type:

  Zooplankton (Copepods) or phytoplankton species

## Value

df of CTI for each sample

## Examples

``` r
df <- pr_get_CTI("Phytoplankton")
```
