# Filter dataframe on `StationCode` to return only NRS stations

Filter dataframe on `StationCode` to return only NRS stations

## Usage

``` r
pr_filter_NRSStations(dat)
```

## Arguments

- dat:

  Dataframe to filter

## Value

Dataframe with only NRS Stations

## Examples

``` r
if (FALSE) dat <- pr_filter_NRSStations(dat) # \dontrun{}
```
