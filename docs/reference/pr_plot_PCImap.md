# Map Phytoplankton Colour Index (PCI) from CPR samples around Australia

Map Phytoplankton Colour Index (PCI) from CPR samples around Australia

## Usage

``` r
pr_plot_PCImap(dat)
```

## Arguments

- dat:

  dataframe with location and seasonal PCI data

## Value

plot of PCI around Australia

## Examples

``` r
pr_get_PCIData() %>% pr_plot_PCImap()
```
