# Get data files available from SOTS moorings

Get data files available from SOTS moorings

## Usage

``` r
pr_get_SOTSvariables(Type = "Physical")
```

## Arguments

- Type:

  Physical or nutrients

## Value

List of variables available from SOTS files

## Examples

``` r
dat <- pr_get_SOTSvariables(Type = "Physical")
```
