# Get STI kernel density for each species

Get STI kernel density for each species

## Usage

``` r
pr_get_STI(Type = "Zooplankton")
```

## Arguments

- Type:

  Zooplankton (Copepods) or phytoplankton species

## Value

df of STI kernel density for each species

## Examples

``` r
df <- pr_get_STI("Phytoplankton")
```
