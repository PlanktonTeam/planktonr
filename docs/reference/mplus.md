# Pipe operator for adding time periods to dates

Pipe operator for adding time periods to dates

## Usage

``` r
e1 %m+% e2
```

## Arguments

- e1:

  A period or a date-time object of class.

- e2:

  A period or a date-time object of class.

## Value

A date-time object of class POSIXlt, POSIXct or Date.
